Pororoca เป็นเครื่องมือทดสอบ HTTP ที่ได้รับแรงบันดาลใจจาก Postman แต่มาพร้อมกับการปรับปรุงมากมาย

รองรับสำหรับ Windows, macOS และ Linux

![ExampleScreen](./misc/example_screen_en.png)

## การติดตั้ง

อ่าน[คำแนะนำ](https://pororoca.io/docs/installation)และดาวน์โหลดได้[ที่นี่](https://github.com/alexandrehtrb/Pororoca/releases)

## คุณสมบัติ

* รองรับ [HTTP/2](https://http2.github.io/) และ [HTTP/3](https://developers.cloudflare.com/http3/)
* กำหนด environment แบบ collection-scoped ได้
* เพิ่มความสะดวกในการจัดการตัวแปร
* ตัวแปรลับ
* คอลเล็กชันและสภาพแวดล้อมสามารถ export ร่วมกันในไฟล์เดียว
* เข้ากันได้อย่างสมบูรณ์กับการ export และ import ไปที่ Postman
* ใช้หน่วยความจำต่ำกว่า Postman สองถึงสามเท่า
* รองรับหลายภาษา
* การทดสอบอัตโนมัติ
* WebSockets
* เปิดแอพพลิเคชันได้เร็วมาก
* ฟรีและโอเพ่นซอร์ส

ตรวจสอบ[เอกสาร](https://pororoca.io/docs/)เพื่อเรียนรู้เพิ่มเติม

*หมายเหตุ*: ใน Windows การรองรับ HTTP/2 ต้องใช้ Windows 10 หรือใหม่กว่า การรองรับ HTTP/3 ต้องใช้ Linux หรือ Windows 11 และใหม่กว่า

### HTTP/2 และ HTTP/3

ต้องการเรียนรู้เพิ่มเติมเกี่ยวกับ HTTP/2 และ HTTP/3? ตรวจสอบ[บทความ](https://alexandrehtrb.github.io/posts/2024/03/http2-and-http3-explained/)นี้

## นโยบายความเป็นส่วนตัว

Pororoca ไม่ซิงค์ข้อมูลผู้ใช้ เช่น ค่ากำหนด, คอลเล็กชัน, สภาพแวดล้อม, ข้อมูลเครื่อง หรือข้อมูลการใช้งาน กับเซิร์ฟเวอร์ระยะไกลใดๆ ค่ากำหนดของผู้ใช้และคอลเล็กชันจะถูกเก็บเป็นไฟล์ในคอมพิวเตอร์ของผู้ใช้
